      • touch

         abstract void touch(float x, float y, int eventType, int pointerId, int width, int height, long timestamp, float orientation, float pressure, float size, float tool_major, float tool_minor, float touch_major, float touch_minor)

        send a detailed touch event with advanced physical attributes. 向云端发送触摸事件。

        Parameters:
        x - The X coordinate of the touch point (relative to the video frame) 触摸点的X坐标（相对于视频帧）
        y - The Y coordinate of the touch point (relative to the video frame) 触摸点的Y坐标（相对于视频帧）
        eventType - Type of touch event (0: DOWN, 1: MOVE, 2: UP) 触摸事件类型 (0: 按下, 1: 移动, 2: 抬起)
        pointerId - Unique identifier for the touch pointer (consistent across events).
        width - Width of the touchable area (video frame width) 可触摸区域的宽度（视频帧宽度）
        height - Height of the touchable area (video frame height) 可触摸区域的高度（视频帧高度）
        timestamp - Timestamp of the touch event (in milliseconds) 触摸事件的时间戳（毫秒）
        orientation - Orientation angle of the touch tool in radians (0 = vertical) 触摸工具的方向角度（弧度，0表示垂直）
        pressure - Normalized pressure value (0.0 = no pressure, 1.0 = max pressure) 标准化压力值（0.0表示无压力，1.
        size - Normalized size of the touch area (0.0 to 1.0) 标准化触摸区域尺寸（0.0到1.
        tool_major - Length of the tool's major axis in pixels 工具主轴长度（像素）
        tool_minor - Length of the tool's minor axis in pixels 工具次轴长度（像素）
        touch_major - Length of the touch contact major axis in pixels 接触区域主轴长度（像素）
        touch_minor - Length of the touch contact minor axis in pixels 接触区域次轴长度（像素）
