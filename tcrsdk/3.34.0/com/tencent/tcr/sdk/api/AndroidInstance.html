public interface AndroidInstance

    AndroidInstance接口定义了云手机实例的各种操作功能。

    该接口提供了对云手机实例的单实例操作和批量操作功能，包括：

    • 截图和文件上传下载
    • 系统设置（分辨率、GPS、传感器等）
    • 应用管理（安装、卸载、启动、停止等）
    • 媒体控制（播放、停止、搜索等）
    • 群控功能（连接、同步等）
    所有操作均通过异步回调返回结果。
