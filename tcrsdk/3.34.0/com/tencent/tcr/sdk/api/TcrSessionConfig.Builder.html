      • enableCustomAudioCapture

         TcrSessionConfig.Builder enableCustomAudioCapture(boolean enableCustomAudioCapture, int inputSampleRate, boolean useStereoInput)

        Sets custom audio capture parameters

        Configures the enablement of custom audio capture, input sample rate, and whether to use stereo input. How to use custom audio capture: Step 0: Make sure your audio data is 16-bit, as currently custom audio capture only supports AudioFormat.ENCODING_PCM_16BIT. Step 1: Call this API to enable custom audio capture, and set the sample rate and number of channels (mono or stereo) for your input audio. Step 2: Use getCustomAudioCaptureDataBufferSize to obtain the right buffer size for allocating space to your ByteBuffer. Step 3: Use setEnableLocalAudio to enable local audio upstream transmission. Step 4: After establishing a connection with TcrSession, call sendCustomAudioData to transmit custom audio data.

        Parameters:
        enableCustomAudioCapture - whether to enable custom audio capture
        inputSampleRate - the sample rate of input audio
        useStereoInput - whether to use stereo input (true: stereo, false: mono)
        Returns:

        Builder instance for method chaining

      • enableCustomVideoCapture

         TcrSessionConfig.Builder enableCustomVideoCapture(boolean enable)

        Enables or disables custom video capture mode

        This method switches between video capture modes: - Disabled (enable = false): Default mode, SDK automatically manages camera capture and video upload - Enabled (enable = true): Custom capture mode, SDK will not start the camera, and you need to actively upload video frames via sendCustomVideo

        Parameters:
        enable - true: enable custom video capture; false: disable custom video capture (use SDK default camera capture)

      • remoteDesktopResolution

         TcrSessionConfig.Builder remoteDesktopResolution(int width, int height)

        Set the resolution of Cloud Desktop. If the cloud PC application is in full-screen mode, the resolution of the downstream video stream will also change accordingly. This method set the desktop resolution at the beginning of the session. If you need to change the desktop resolution during run time, please use the function setRemoteDesktopResolution. Some decoders on certain devices may fail to decode videos at certain resolutions. You can use com.tencent.tcr.sdk.api.utils.VideoCapabilityUtil to query the recommended compatible resolution for your device. NOTE：This method is only for PC application, and the mobile application is not supported.

        Parameters:
        width - the width of cloud desktop, which must be an even value.
        height - the height of cloud desktop.
