## 一、功能介绍：
腾讯云渲染VR Unity Plugin是一个基于OpenXR Plugin实现的插件，这个插件允许您将腾讯云渲染功能集成到您的VR程序中。

它能够自动获取一体机的头显位姿、手柄位姿、手柄按键等事件，并将这些事件发送到云渲染实例。同时，它还能够持续渲染云端实例下发的视频画面。
